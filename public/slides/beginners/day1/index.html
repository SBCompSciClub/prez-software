What is web developement?

Programming which consists of developing apps for the web such as websites, PWA (progressive web apps) and games using the web languages: HTML, CSS & JS.

What is HTML?

  • Stands for - Hyper Text Markup Language
  • Used to create web pages
  • Primaraly used to create Static Content
  • Styling is done using CSS, which we will talk about later

More about HTML

  • HTML is dumb. It has no logic.
  • HTML is made up of 'tags'
  • An opening tag looks like 
    
								<tagname>
  • While a closing tag looks like 
    
												</tagname>
  • Put them together:
  • Most opening tags are always followed by a closing tag

    • A simple HTML boilerplate

    
						    <html>
							  <head>
							    <title> </title>
							    </head>
						      <body>
							  </body>
							</html>

    Headings and Paragraphs

    Heading: <h1>
    Paragraph: <p>

    Tags within Tags


    • Correct Embedding -> <h1><b> Hello </b> </h1>
    • Close the innermost tag first, then the outermost.
    • Incorrect Embedding -> <h1> <b> Hello </h1> </b>

    Images in HTML


    • <img src = "url" width = "width in px or % " height = "height in px or %">
